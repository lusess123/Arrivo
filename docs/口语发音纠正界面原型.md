## 交互说明

### 手势操作

| 操作 | 触发 | 说明 |
|------|------|------|
| 按住说话 | 长按麦克风按钮 | 开始录音 |
| 松开发送 | 释放按钮 | 发送语音 |
| 上滑取消 | 按住时上滑 | 取消本次录音 |

### 动效说明

| 场景 | 动效 |
|------|------|
| AI回复中 | 气泡渐入 + 打字机效果 |
| 评分展示 | 数字从0滚动到目标分数 |
| 进度条 | 平滑填充动画 |
| 纠错提示 | 轻微震动 + 高亮闪烁 |

### 错误状态

| 场景 | 处理 |
|------|------|
| 网络断开 | Toast提示 + 重试按钮 |
| 录音失败 | 弹窗引导开启麦克风权限 |
| 分析超时 | 显示"稍后查看报告" |
